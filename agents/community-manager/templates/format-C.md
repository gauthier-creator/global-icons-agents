# Format C — Decryptage Pratique

## Usage
- **Jours** : Mardi (principal), Jeudi (selon actualite)
- **Ton** : Pedagogique, precis, actionnable
- **Longueur** : 200-400 mots

## Structure

### 1. Hook (1 ligne)
Question pratique que se pose un CGP/FO.
Patterns recommandes :
- "Comment {action pratique} ? Guide en 4 points."
- "{Sujet technique} : ce que tout CGP devrait savoir."
- "Vos clients vous demandent {question}. Voici comment repondre."
- "{Mecanisme financier} explique en 3 minutes."

### 2. Le probleme / la question (2-3 phrases)
- Poser le contexte pratique
- Pourquoi c'est important MAINTENANT
- A qui ca s'adresse (CGP, FO, ou les deux)

### 3. L'explication du mecanisme (5-8 phrases)
- Decrypter le sujet de facon claire et structuree
- Utiliser des bullet points si necessaire
- Chaque point appuye par un exemple ou un chiffre
- Vocabulaire technique maitrise mais accessible
- Peut inclure une mini-liste ou un comparatif

### 4. Exemple concret chiffre (2-3 phrases)
- Un cas d'usage reel ou realiste
- Avec des chiffres concrets (montants, rendements, durees)
- Montrer le resultat pratique

### 5. Points de vigilance (2-3 phrases)
- Ce qu'il faut surveiller / les pieges a eviter
- Nuancer sans decourager
- Montrer l'expertise

### 6. CTA pratique (1-2 phrases)
- Question pratique invitant au partage d'experience
- Ou checklist/outil utile

## Exemple de structure

```
{Question pratique accrocheuse}

{Contexte : pourquoi cette question se pose maintenant.
A qui ca s'adresse et pourquoi c'est important.}

Voici comment ca fonctionne :

📌 {Point 1 — mecanisme explique clairement}
📌 {Point 2 — avec chiffre ou comparaison}
📌 {Point 3 — implication pratique}
📌 {Point 4 — si necessaire}

Exemple concret :
{Cas chiffre realiste montrant le mecanisme en action.
Resultat tangible pour un CGP/FO.}

⚠️ Points de vigilance :
{Ce qu'il faut surveiller.}
{Piege courant a eviter.}

{CTA pratique — question d'experience ou checklist}

#DecryptageCGP #GestionDePatrimoine #Alternatives
```

## Checklist
- [ ] Hook question pratique
- [ ] Contexte clair et actuel
- [ ] Explication structuree (points ou etapes)
- [ ] Au moins 1 exemple concret chiffre
- [ ] Points de vigilance (nuance)
- [ ] CTA pratique (pas commercial)
- [ ] 200-400 mots
- [ ] Zero mention Global Icons
- [ ] Ton pedagogique sans etre condescendant
- [ ] Actionnable : le lecteur repart avec quelque chose
