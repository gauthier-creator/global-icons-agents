# Format B — Insight Contre-Intuitif

## Usage
- **Jours** : Mercredi (principal), Jeudi ou Vendredi (selon actualite)
- **Ton** : Provocant, rigoureux, surprenant
- **Longueur** : 150-350 mots

## Structure

### 1. Hook (1 ligne)
Affirmation contre-intuitive qui cree un arret de scroll.
Patterns recommandes :
- "Non, {idee recue} n'est pas {ce qu'on croit}."
- "{Affirmation choquante}. Voici pourquoi."
- "Tout le monde pense que {X}. Les donnees montrent l'inverse."
- "Et si {croyance commune} etait completement faux ?"

### 2. L'idee recue (2-3 phrases)
- Enoncer clairement ce que "tout le monde" croit
- Montrer pourquoi cette croyance est repandue
- Pas de jugement : juste le constat

### 3. Les donnees qui prouvent le contraire (3-5 phrases)
- Presenter les chiffres qui invalident l'idee recue
- Citer la source explicitement
- Etre factuel et rigoureux (pas d'exageration)
- Si possible, comparer deux periodes ou deux actifs

### 4. Implications pour CGP/FO (2-3 phrases)
- Qu'est-ce que ca change concretement dans leur pratique ?
- Quelle decision d'allocation cela remet en question ?
- Un conseil implicite (sans etre commercial)

### 5. Question ouverte (1-2 phrases)
- Inviter a la reflexion ou au debat
- Encourager les commentaires

## Exemple de structure

```
{Affirmation contre-intuitive forte.}

La croyance : {ce que la plupart des professionnels pensent}.
{Pourquoi cette idee est si repandue — 1-2 phrases.}

La realite :
{Donnee 1 qui contredit — source citee.}
{Donnee 2 qui renforce — comparaison.}
{Donnee 3 si disponible.}

Pour un CGP/FO, ca signifie :
{Implication pratique concrete.}
{Ce que ca change dans l'approche d'allocation.}

{Question ouverte qui suscite le debat}

#Alternatives #InvestissementAlternatif #CGP
```

## Checklist
- [ ] Hook contre-intuitif qui cree la surprise
- [ ] Idee recue clairement enoncee
- [ ] Au moins 2 donnees sourcees qui la contredisent
- [ ] Pas d'exageration — rigueur factuelle
- [ ] Implications pratiques pour CGP/FO
- [ ] Question ouverte finale
- [ ] 150-350 mots
- [ ] Zero mention Global Icons
- [ ] Ton surprenant mais jamais condescendant
