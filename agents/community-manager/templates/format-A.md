# Format A — Chronique Marche

## Usage
- **Jours** : Lundi, Vendredi (ou Jeudi selon actualite)
- **Ton** : Analytique, factuel, expert
- **Longueur** : 200-400 mots

## Structure

### 1. Hook (1 ligne)
Chiffre surprenant ou constat de marche inattendu.
Patterns recommandes :
- "{Chiffre}% : c'est la {metrique} qui vient de {mouvement}."
- "Le marche {X} vient de franchir un seuil que personne n'attendait."
- "Les derniers chiffres {source} viennent de tomber. Et ils changent la donne."

### 2. Contextualisation macro (2-3 phrases)
- Situer le chiffre dans le contexte macro
- Expliquer pourquoi c'est significatif maintenant
- Lier a une tendance plus large

### 3. Analyse des implications (3-5 phrases)
- Impact sur les investissements alternatifs specifiquement
- Ce que ca signifie pour l'allocation d'un CGP/FO
- Comparaison avec d'autres classes d'actifs si pertinent
- Au moins 1 donnee supplementaire sourcee

### 4. Donnee cle (1-2 phrases)
- Le chiffre le plus marquant, mis en perspective
- Source explicitement citee

### 5. CTA / Question finale (1-2 phrases)
- Question ouverte qui suscite la reflexion
- Ou invitation a commenter/partager une experience

## Exemple de structure

```
📊 {Chiffre accrocheur lié au marché}.

{Contextualisation : pourquoi ce chiffre est important aujourd'hui.
Lien avec la tendance macro en cours.}

{Analyse : ce que ca implique pour les alternatifs.
Impact concret sur l'allocation d'actifs d'un CGP ou FO.
Comparaison avec une autre classe d'actifs pour illustrer.}

{Donnee complementaire sourcee pour renforcer l'argument.}

{Source : {nom_source}}

{Question finale ou CTA engageant}

#Alternatives #CGP #GestionDePatrimoine
```

## Checklist
- [ ] Hook avec chiffre ou constat marche
- [ ] Contextualisation macro claire
- [ ] Lien explicite avec les alternatifs
- [ ] Au moins 2 donnees chiffrees
- [ ] Source(s) citee(s)
- [ ] Implication pratique pour CGP/FO
- [ ] Question ou CTA final
- [ ] 200-400 mots
- [ ] Zero mention Global Icons
